Prompt:
How do I deploy a Python + Celery + Redis + PostgreSQL app to Render on free tier?

AI Output:
- Create Render Web Service for FastAPI/Flask app
- Create Render Background Worker for Celery
- Use Render Redis instance
- Use Render PostgreSQL instance
- Put environment variables:
  DATABASE_URL
  REDIS_URL
  CELERY_BROKER_URL
  CELERY_RESULT_BACKEND
- Build command:
  pip install -r requirements.txt
- Start commands:
  API: uvicorn app.main:app --host 0.0.0.0 --port 10000
  Worker: celery -A app.workers.celery_app worker --loglevel=info
